WoTui Cookie 助手v0.4.0

在物理机浏览器导出 m.weibo.cn（移动版微博）Cookie，一键推送到 Docker 实例
⚠️
必须使用 m.weibo.cn（移动版），不要用 weibo.com（PC 版）。
PC 版有强反爬机制，Cookie 即使保存也会被服务端拦截。WoTui 后端验证接口走的也是 m.weibo.cn/api/config。

1从浏览器获取 m.weibo.cn Cookie

展开 / 收起获取步骤
方法 A（最推荐 · 一行搞定）
  1. 在浏览器中打开 https://m.weibo.cn 并完成登录（PC 浏览器也可直接打开，页面是移动版样式）
  2. 按 F12（Mac 上为 ⌥⌘I）打开开发者工具
  3. 切到 Console 标签，输入 document.cookie 回车
  4. 复制返回的完整字符串（去掉首尾引号即可），粘贴到下方步骤 2
方法 B（DevTools Application 面板）
  1. 打开 DevTools → Application → 左侧 Cookies → 选择 https://m.weibo.cn
  2. 至少复制 SUB、SUBP、_T_WM、MLOGIN 四项
  3. 按 name=value; name=value 拼接粘贴下方
方法 C（浏览器扩展导出 JSON）
  1. 安装 EditThisCookie / Cookie-Editor 类扩展
  2. 在 m.weibo.cn 页面上点扩展图标 → Export
  3. 粘贴下方，本工具自动识别 JSON 格式

4导入到 WoTui

localhost:3000 127.0.0.1:3000 host.docker.internal:3000
复制下方 Cookie 字符串，粘贴到 WoTui 网页登录弹窗的「手动粘贴 Cookie」标签。
本工具完全运行在本地浏览器，不会上传任何数据到第三方。
所有 HTTP 请求都直接发往你填写的 Docker URL（默认 localhost:3000）。
Cookie 域默认设为 .weibo.cn，与后端验证接口 m.weibo.cn/api/config 一致。
